# special variables

### Examples of special variables

#### `$_`

Holds the value of the last argument of the last command executed

#### `$0`

Holds the name of the currently active shell or script (refer to  [https://jarrettgxz-sec.gitbook.io/linux/general/environment-variables](https://jarrettgxz-sec.gitbook.io/linux/general/environment-variables) `$SHELL`section for examples)
